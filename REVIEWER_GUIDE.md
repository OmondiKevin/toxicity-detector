# Reviewer Guide

This package contains:
- Trained models (if included) under `models/`
- Processed datasets under `data/processed/`
- App code under `app/`
- Scripts (optional) under `scripts/`

### What to verify
1. App launches and loads models.
2. Basic inference works on sample inputs.
3. Reproducible predictions using provided test files.

### Expected Outputs
- The app UI should load without additional configuration.
- Prediction results should be consistent across runs.
